# Institucional

Propósito: apresentar missão, contexto, áreas de pesquisa e integrantes a partir da fonte única em `shared/config/institutional.ts`.

Limites: não armazena dados pessoais sem autorização e não duplica conteúdo entre Home e `/sobre`.

Entradas e saídas: recebe dados institucionais estruturados e expõe grades de áreas e membros para composição em `app`.
