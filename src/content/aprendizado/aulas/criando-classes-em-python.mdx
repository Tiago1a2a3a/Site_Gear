
## Atributos, métodos e self

O método `__init__` prepara o estado inicial. `self` representa o próprio objeto, enquanto os métodos definem seus comportamentos.

```python
class Robo:
    def __init__(self, nome, bateria=100):
        self.nome = nome
        self.bateria = bateria

    def mover(self, distancia):
        consumo = distancia * 2
        if consumo > self.bateria:
            return "Bateria insuficiente."
        self.bateria -= consumo
        return f"{self.nome} moveu {distancia} metros."

robo = Robo("Luna", 80)
print(robo.mover(10))
```
