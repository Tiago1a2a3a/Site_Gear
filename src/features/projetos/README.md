# Projetos

Propósito: apresentar os projetos de robótica do GEAR e seus recursos opcionais.
A feature permanece independente da área de aprendizado.

`data/projetos.ts` expõe a coleção tipada em ordem alfabética estável. A Home
consome apenas entradas com `destaque: true`, mantém a mesma ordenação e limita a
vitrine aos três primeiros projetos. Alterar esse limite ou a regra de ordem é
uma decisão editorial e deve ser documentada aqui.

Imagens, vídeos, tecnologias, GitHub e documentação são opcionais. Componentes
não devem renderizar títulos ou blocos vazios quando esses campos não existirem.
Links externos abrem em nova aba com indicação acessível e `rel="noreferrer"`.

Manutenção: responsabilidade atual de @Tiago1a2a3a; revise a Parte A antes de
alterar.
